A simple API to fetch merchant data. Returns either a set of div containing spans of information, an encoded JSON string, or a simple string formatted @slot a name¦slot b name¦slot c name@ .

Usage

Data for today 
{{Travelling Merchant/api}}

Fetching this via API

You'll want to query the wiki's API - send a GET request to https://runescape.wiki/api.php with query 

format=json
action=parse
prop=text
disablelimitreport=1
text={{Travelling%20Merchant/api}}

With text equal to the template usage you want. Examples:

  • Full, today: https://runescape.wiki/api.php?format=json&action=parse&prop=text&disablelimitreport=1&text={{Travelling%20Merchant/api}}
  • Simple, in 5 days: https://runescape.wiki/api.php?format=json&action=parse&prop=text&disablelimitreport=1&text={{Travelling%20Merchant/api%7Coffset=5%7Cformat=simple}}
  • JSON, for yesterday: https://runescape.wiki/api.php?format=json&action=parse&prop=text&disablelimitreport=1&text={{Travelling%20Merchant/api%7Coffset=-1%7Cformat=json}}
